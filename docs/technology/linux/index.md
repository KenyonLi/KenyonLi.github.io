## linux 
::: tip linux  下使用 vim 卡死原因与解决方法

> 我们在[linux](https://so.csdn.net/so/search?q=linux&spm=1001.2101.3001.7020) 下使用vim 编辑文件时，习惯性的使用了 Ctrl + s ， 导致vim 僵死，无法编辑，此时只需要使用 Ctrl + q 解除即可。
>
>  
>
> 在vim 中，Ctrl + S 是锁屏快捷键，  Ctrl+ q 解锁 

:::
